Personal Finance Manager

A Python console application for managing personal finances and tracking income and expenses.

## Project Description
This project allows users to view and budget their monthly expenses. 

## Project Files

### Current Version (v1.1 Beta)
- `budget.py` - Contains the main Budget class for OOP structure, file saving, and tracking
- `main_file.py` - The main program to run version 1.1 Beta

### Previous Version (v1.0 Alpha - Preserved)
- `v1_0_alpha.py` - The original version of the app using dictionaries (originally from finance_manager.py)

## Version History

### v1.1 Beta (Current/In Development)
- Original v1.0 features still maintained
- Budget class Implemented
- Saves & loads budget data to JSON files
- Tracks daily expenses against monthly limits
- In Progress: Budget reporting features

### v1.0 Alpha 
- Basic monthly budget tracking using dictionaries
- Input for income, fixed costs, and spending limits  
- Track grocery and miscellaneous expenses
- Automatic balance updates and budget alerts

## How to Run
```bash
# Run the new version v1.1 Beta
py main_file.py

# Run the old version
py v1_0_alpha.py
