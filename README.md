Personal Finance Manager

A Python console application for managing personal finances and tracking income and expenses.

## Project Description
This project allows users to view and budget their monthly expenses.

## Version History

### v1.0 Alpha (Current)
- Basic monthly budget tracking using dictionaries
- Input for income, fixed costs, and spending limits  
- Track grocery and miscellaneous expenses
- Automatic balance updates and budget alerts
- Uses nested dictionaries to organize financial data

### v1.1 Beta (Planned: est.Dec 2026)
- Rewrite using classes for better organization
- Track daily spending against monthly budget
- Keep budget data when you close the program (Files)
- Better monthly reports and spending history
- Compare planned vs actual spending over time

## How to Run
```bash
py finance_manager.py
